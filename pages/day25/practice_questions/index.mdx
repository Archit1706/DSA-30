# Two Pointers Practice Questions

Ten interview classics, every flavor represented. Each problem is a direct instance of one of the [opposite-ends](/day25/opposite_ends), [same-direction](/day25/same_direction), or [fast-and-slow](/day25/fast_slow) templates — once you spot which flavor it is, the code writes itself.

<Callout type="info">
    Before reading any solution, force yourself through the four-question checklist:

    1. **Which flavor?** (Opposite-ends / same-direction / fast-and-slow)
    2. **What does each pointer represent?**
    3. **What's the movement rule?**
    4. **When do I stop?**

    The code is six to ten lines. The whole chapter.
</Callout>
