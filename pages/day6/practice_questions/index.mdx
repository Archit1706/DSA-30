# Tree Practice Questions

Four classic tree problems. Each one is a small variation on the **three-question recipe** from [Day 5 — Thinking Recursively](/day5/thinking_recursively): base case for null, recursive call on each child, combine.

<Callout type="info">
    Notice how *short* the solutions are — most of these are 3–5 lines. That's the magic of recursion on trees: you describe one level, and the call stack handles the entire structure.
</Callout>
