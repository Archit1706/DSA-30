### Sum of Natural Numbers

    #### Code

    <Tabs items={['C++', 'Python']} defaultIndex="0">
        <Tab>
        ```cpp copy
        int sum(int n) {
            if (n == 1) return n;
            return n + sum(n - 1);
        }
        ```

        </Tab>

        <Tab>
        ```python copy
        def sum(n):
            if n == 1:
                return n
            return n + sum(n - 1)
        ```

        </Tab>

        #### Explanation

        1. Base case: `n == 1`
        2. Recursive case: `n + sum(n - 1)`

        #### Analysis

        - Time Complexity: `O(n)`
        - Space Complexity: `O(n)`
    </Tabs>

    ### Decimal to Binary COnversion

    #### Code

    <Tabs items={['C++', 'Python']} defaultIndex="0">
        <Tab>
        ```cpp copy
        int decimalToBinnary(int num) {
            if (num == 0) return;
            decimalToBinnary(num / 2);
            cout << num % 2;
        }
        ```

        </Tab>

        <Tab>
        ```python copy
        def decimal_to_binary(num):
            if num == 0:
                return
            decimal_to_binary(num // 2)
            print(num % 2, end='')
        ```

        </Tab>

        #### Explanation

        1. Base case: `n == 0`
        2. Recursive case: `decimalToBinnary(num / 2)`
        3. Print the remainder of `num / 2`

        #### Analysis

        - Time Complexity: `O(log n)`
        - Space Complexity: `O(log n)`
    </Tabs>

    ### Count Vowels in a String

    #### Code

    <Tabs items={['C++', 'Python']} defaultIndex="0">
        <Tab>
        ```cpp copy
        int countVowels(string str, int start, int end) {
            if (start >= end) return 0;
            int count = countVowels(str, start + 1, end);
            if (str[start] == 'a' || str[start] == 'e' || str[start] == 'i' || str[start] == 'o' || str[start] == 'u') {
                count++;
            }
            return count;
        }
        ```

        </Tab>

        <Tab>
        ```python copy
        def count_vowels(str, start, end):
            if start >= end:
                return 0
            count = count_vowels(str, start + 1, end)
            if str[start] == 'a' or str[start] == 'e' or str[start] == 'i' or str[start] == 'o' or str[start] == 'u':
                count += 1
            return count
        ```

        </Tab>

        #### Explanation

        1. Base case: `start >= end`
        2. Recursive case: `countVowels(str, start + 1, end)`
        3. Return `0` when `start >= end`
        4. Return `count + 1` when `str[start]` is a vowel

        #### Analysis

        - Time Complexity: `O(n)`
        - Space Complexity: `O(n)`
    </Tabs>

    ### Sum of numbers in a Singly Linked List

    #### Code

    <Tabs items={['C++', 'Python']} defaultIndex="0">
        <Tab>
        ```cpp copy
        int sumNodes(Node* head) {
            if (head == NULL) return 0;
            return head->data + sumNodes(head->next);
        }
        ```

        </Tab>

        <Tab>
        ```python copy
        def sum_nodes(head):
            if head == None:
                return 0
            return head.data + sum_nodes(head.next)
        ```

        </Tab>

        #### Explanation

        1. Base case: `head == NULL`
        2. Recursive case: `sumNodes(head->next)`
        3. Return `0` when `head == NULL`
        4. Return `head->data + sumNodes(head->next)` when `head != NULL`

        #### Analysis

        - Time Complexity: `O(n)`
        - Space Complexity: `O(n)`
