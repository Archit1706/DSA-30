# D&C Practice Questions

Ten interview classics, every D&C shape represented. Each problem is a direct application of one of the [four canonical recurrence forms](/day26/the_template) or an [advanced pattern](/day26/advanced) — once you spot which shape it is, the code writes itself.

<Callout type="info">
    Before reading any solution, force yourself through the four-question checklist:

    1. **How do I split the input?**
    2. **What's the base case?**
    3. **What's the combine step?**
    4. **What's the recurrence and the runtime?** (Use the [Master Theorem](/day26/master_theorem).)

    The code is a 10-line transcription of those four answers. If you can write the recurrence, you've already solved the analysis.
</Callout>
