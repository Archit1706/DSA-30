### Stack using Linked List

    <Steps>
        ### Insert an Element

        - ALGORITHM

        ```cpp copy
        Step 1: Allocate memory for the new node and name it newNode
        Step 2: SET newNode->data = val
        Step 3: if top = NULL
                    SET newNode->next = NULL
                    SET top = newNode
                else
                    SET newNode->next = top
                    SET top = newNode
                [END OF if]
        Step 4: END
        ```
