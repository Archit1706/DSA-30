## Basic Operations on a Singly Linked List

> We can perform the following basic operations on a Singly Linked List:

1. Define a Linked List
2. Insert an Element at the beginning
3. Insert an Element at the end
4. Insert an Element after a given position
5. Insert an Element before a given position
6. Delete an Element at the beginning
7. Delete an Element at the end
8. Delete an Element after a given position
9. Delete an Element before a given position
10. Count the number of Elements/Nodes
11. Traverse and Display the Elements/Nodes
12. Reverse the Linked List

## Complexity at a Glance

<ComplexityTable operations={[
    { name: "Access by index (traverse to position k)", time: "O(n)", space: "O(1)", notes: "No random access — must follow pointers" },
    { name: "Search (find by value)", time: "O(n)", space: "O(1)", notes: "Must scan from HEAD in worst case" },
    { name: "Insert at beginning", time: "O(1)", space: "O(1)", notes: "Just update HEAD pointer" },
    { name: "Insert at end (no tail ptr)", time: "O(n)", space: "O(1)", notes: "Must traverse to find last node" },
    { name: "Insert at end (with tail ptr)", time: "O(1)", space: "O(1)", notes: "Directly update TAIL pointer" },
    { name: "Insert at position k", time: "O(n)", space: "O(1)", notes: "Traverse to k-1, then rewire pointers" },
    { name: "Delete at beginning", time: "O(1)", space: "O(1)", notes: "Move HEAD to HEAD→next" },
    { name: "Delete at end", time: "O(n)", space: "O(1)", notes: "Traverse to second-to-last node" },
    { name: "Delete at position k", time: "O(n)", space: "O(1)", notes: "Traverse to k-1, bypass node k" },
    { name: "Count nodes", time: "O(n)", space: "O(1)", notes: "Single pass from HEAD to NULL" },
    { name: "Reverse", time: "O(n)", space: "O(1)", notes: "Three-pointer iterative approach" },
]} />

<Callout type="info">
    The key insight: **insertion and deletion themselves are O(1)** once you
    have a pointer to the right position. The O(n) cost comes from **finding**
    that position, not from the rewiring of pointers.
</Callout>

    ### Insert an Element at the End

    - ALGORITHM

    ```py copy
    Step 1: if AVAIL = NULL
                Write Overflow
                Go to step 10
            [END OF if]
    Step 2: SET newNode = AVAIL
    Step 3: SET AVAIL = AVAIL->next
    Step 4: SET newNode->data = val
    Step 5: SET newNode->next = NULL
    Step 6: SET last = start
    Step 7: Repeat Step 8 while last->next != NULL
    Step 8:         SET last = last->next
            [END OF LOOP]
    Step 9: SET last->next = newNode
    Step 10: EXIT
    ```
    - CODE

        ```cpp copy
            if (start == nullptr)
                start = new_node;
            else
            {
                Node *temp = start;
                while (temp->next != nullptr)
                    temp = temp->next;

                temp->next = new_node;
                new_node->next = nullptr;
            }
            return;
        }
        ```

        ```py copy
                self.start = newNode

                temp = self.start
                while temp.next is not None:
                    temp = temp.next
                temp.next = newNode
        ```

    ### Delete an Element before a given position

    - ALGORITHM

    ```py copy
    Step 1: if start = NULL or start->next = NULL
                Write Underflow (list too short)
                Go to step 12
            [END OF if]
    Step 2: if start->next->data = pos
                Write "No element before pos"
                Go to step 12
            [END OF if]
    Step 3: SET ptr = start->next
    Step 4: SET prePtr = start
    Step 5: SET prePre = NULL
    Step 6: Repeat Steps 7 to 9 while ptr->data != pos
    Step 7:         SET prePre = prePtr
    Step 8:         SET prePtr = ptr
    Step 9:         SET ptr = ptr->next
            [END OF LOOP]
    Step 10: SET prePre->next = ptr   (bypass prePtr)
    Step 11: free prePtr
    Step 12: EXIT
    ```

    - CODE

    <Tabs items={["C++", "Python"]} defaultIndex="0">
        <Tab>
        ```cpp copy
        void delete_before(int pos) {
            if (start == nullptr || start->next == nullptr) {
                cout << "List is empty or has only one node!" << endl;
                return;
            }

            if (start->data == pos) {
                cout << "No element found before the specified position!" << endl;
                return;
            }

            if (start->next->data == pos) {
                Node *temp = start;
                start = start->next;
                delete temp;
                return;
            }

            Node *ptr = start->next;
            Node *pre_ptr = start;
            Node *pre_pre = nullptr;

            while (ptr != nullptr && ptr->data != pos) {
                pre_pre = pre_ptr;
                pre_ptr = ptr;
                ptr = ptr->next;
            }

            if (ptr == nullptr) {
                cout << "No such element found!" << endl;
                return;
            }

            pre_pre->next = ptr;
            delete pre_ptr;
        }
        ```
        </Tab>
        <Tab>
        ```py copy
        def deleteBefore(self, pos):
            if self.start is None or self.start.next is None:
                print("List is too short!")
                return
            if self.start.next.data == pos:
                self.start = self.start.next
                return
            temp = self.start
            while temp.next.next is not None and temp.next.next.data != pos:
                temp = temp.next
            if temp.next.next is None:
                print("No such element found!")
                return
            temp.next = temp.next.next
        ```

    ### Reverse a Linked List

    - ALGORITHM

    ```py copy
    Step 1: SET current = start
    Step 2: SET prev = NULL
    Step 3: SET nextPtr = NULL
    Step 4: Repeat steps 5 to 7 while current is not NULL
    Step 5:     SET nextPtr = current->next
    Step 6:     SET current->next = prev
    Step 7:     SET prev = current
                SET current = nextPtr
            [END OF LOOP]
    Step 8: SET start = prev
    Step 9: EXIT
    ```

    <Callout type="info">
        This is called the **three-pointer technique**. At each step: save `next`,
        flip the arrow backward (`current→prev`), then advance both pointers. After
        the loop, `prev` points to the new head.
    </Callout>

    - CODE

    <Tabs items={["C++", "Python"]} defaultIndex="0">
        <Tab>
        ```cpp copy
        void reverse()
        {
            if (start == nullptr)
            {
                cout << "Empty List!" << endl;
                return;
            }
            Node *prev_ptr = nullptr, *current_ptr = start, *next_ptr = nullptr;
            while (current_ptr != nullptr)
            {
                next_ptr = current_ptr->next;
                current_ptr->next = prev_ptr;
                prev_ptr = current_ptr;
                current_ptr = next_ptr;
            }
            start = prev_ptr;
        }
        ```
        </Tab>
        <Tab>
        ```py copy
        def reverse(self):
            if self.start == None:
                print("The Linked list is Empty!")
                return
            prevPtr = None
            currentPtr = self.start
            nextPtr = None

            while (currentPtr != None):
                nextPtr = currentPtr.next
                currentPtr.next = prevPtr
                prevPtr = currentPtr
                currentPtr = nextPtr

            self.start = prevPtr
        ```
        </Tab>
    </Tabs>

<QuizBox

    options={["O(1)", "O(log n)", "O(n)", "O(n²)"]}

    explanation="To delete the last node you need to find the second-to-last node (so you can set its next to NULL). Starting from HEAD, you must traverse n-1 nodes — that's O(n).
