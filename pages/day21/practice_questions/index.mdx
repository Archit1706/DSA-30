# Shortest Paths Practice Questions

Eight interview classics covering every algorithm in the chapter. Each problem is a direct instance of [BFS](/day21/bfs_shortest_path), [Dijkstra](/day21/dijkstra), [Bellman-Ford](/day21/bellman_ford), or [Floyd-Warshall](/day21/floyd_warshall) — once you recognize which one, the code writes itself.

<Callout type="info">
    Before reading any solution, force yourself through the [decision flow](/day21/picking_the_right_one):

    1. **Equal weights?** → BFS
    2. **Single source, non-negative?** → Dijkstra
    3. **Negative edges?** → Bellman-Ford
    4. **All pairs, small V?** → Floyd-Warshall
    5. **Constraint on edge count?** → modified Bellman-Ford / expanded-state Dijkstra

    Five questions. Four algorithms. The whole chapter.
</Callout>
