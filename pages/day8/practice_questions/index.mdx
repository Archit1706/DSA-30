# Hash Map Practice Questions

Five interview problems that all collapse to the same idea: **"if I knew whether I'd seen this before, the answer would be one line."** The hash map is what makes "knowing" O(1).

<Callout type="info">
    Two of these problems — [Two Sum](/day2/arrays/practice_questions/two_sum) and [Top K Frequent Elements](/day7/practice_questions/top_k_frequent) — also live in earlier chapters where they're the canonical example. Same pattern, just told from a different angle.
</Callout>
