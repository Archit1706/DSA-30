# Hash Map Practice Questions

Ten interview problems that all collapse to the same idea: **"if I knew whether I'd seen this before, the answer would be one line."** The hash map is what makes "knowing" O(1).

<Callout type="info">
    Two related problems live in earlier chapters: [Two Sum](/day2/arrays/practice_questions/two_sum) (which is the canonical example) and [Top K Frequent Elements](/day7/practice_questions/top_k_frequent) (which combines a counter with a heap). Same pattern, different angles.
</Callout>
